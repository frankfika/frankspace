# Public 文件夹

## 简历文件

请将你的简历 PDF 文件放在这个目录下：

### 中文简历
- 文件名：`resume-zh.pdf`
- 下载文件名：`陈方简历.pdf`

### 英文简历
- 文件名：`resume-en.pdf`
- 下载文件名：`Frank-Chen-Resume.pdf`

## 使用方法

1. 将你的中文简历 PDF 重命名为 `resume-zh.pdf`
2. 将你的英文简历 PDF 重命名为 `resume-en.pdf`
3. 复制这两个文件到当前目录（`/public`）
4. 重启开发服务器

主页上的下载按钮会自动使用这些文件。

## 其他静态资源

你也可以将其他静态资源（图片、字体等）放在这个目录，它们会被直接serve在根路径下。
